Air Hockey 2099

Canvas not supported

To first start the game press the 'Spectate Game' button. Player is red and will use the arrow keys. Player 2 is blue and will use the 'WASD' keys. When a player wants to leave, depending on the player press the appropriate 'Give up Player ' button. If there are already two players playing and you want to watch, also press the 'Spectate Game' button, however you will only see the game live once contact with the paddle has happened.
